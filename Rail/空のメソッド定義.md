## 空のメソッドに対するRailsの補完

例として、空のメソッドを次のように定義したとする
```
class UsersController < ApplicationController

  def profile;end

end
```

Railsはrenderメソッドを補って動作してくれる

```
class UsersController < ApplicationController

  def profile
    render profile
    #users/profile.html.erbをrenderする
  end

end
```

URLカスタマイズするときに使ったりする
