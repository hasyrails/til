## git fetch - リモートリポジトリの内容を確認する -

### fetch ?

> `主な意味` 取ってくる、呼んでくる、(行って)取ってくる、連れてくる

### git pull との違い

> pullを実行すると、リモートリポジトリの内容のマージが自動的に行われてしまいます。
> しかし、単にリモートリポジトリの内容を確認したいだけの時はマージをしたくない場合もあります。
> そのような時はfetchを使用します。

> fetchを実行すると、リモートリポジトリの最新の履歴の取得***だけ***を行うことができます。
