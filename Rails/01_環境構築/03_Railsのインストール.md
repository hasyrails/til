### Railsのインストール
Railsのインストール
```
$ gem install rails -v [Rails version]
```

Railsのバージョン確認
```
$ rails -v
```

### javascriptを使用する場合

javascriptを使用する場合、Javascriptランタイムをインストールする必要がある
