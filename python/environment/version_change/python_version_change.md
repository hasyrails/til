### Pythonのversionを切り替える
#### pyenvを使う場合

##### カレントディレクトリのみpythonのversionを変更する
```
$ pyenv local 3.8.3
```
##### 全てのディレクトリのpythonのversionを変更する
```
$ pyenv global 3.8.3
```
